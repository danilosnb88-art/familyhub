# FamilyHub

FamilyHub is a learning project where I am building a simple web application to help families discover local events.

## Purpose
This project is part of my learning journey into web development and cybersecurity fundamentals. I use it to practice HTML, CSS, debugging, and problem-solving skills.

## What I built so far
- Homepage layout using HTML and CSS
- Navigation bar and hero section
- Event filtering UI (age, season, cost, family size)
- Basic event card structure

## Status
This project is currently **in progress**. I am actively improving layout, styling consistency, and planning to add interactivity.

## Skills I am practicing
- HTML structure and semantic layout
- CSS styling and UI design
- Debugging and fixing layout issues
- Iterative development and problem-solving

## Future improvements
- Responsive design
- JavaScript interactivity (filters/search)
- UI/UX improvements
